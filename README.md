# Osteoporosis-Risk-Prediction-using-Machine-Learning
This project develops a machine learning-based osteoporosis risk prediction system using demographic, lifestyle, medical history, hormonal, and nutritional factors.

The project covers the complete machine learning lifecycle, including data cleaning, exploratory data analysis, feature engineering, model development, hyperparameter tuning, and model evaluation.

Features
Data Cleaning & Missing Value Handling
Exploratory Data Analysis (EDA)
Feature Engineering
Data Preprocessing Pipelines
Class Imbalance Handling (SMOTE)
Multiple Machine Learning Models
Hyperparameter Optimization using GridSearchCV
Performance Evaluation with ROC-AUC and Classification Metrics
Feature Importance Analysis
Models Used
Logistic Regression
Decision Tree Classifier
Random Forest Classifier
Support Vector Machine (SVM)
XGBoost Classifier
Technologies
Python
Pandas
NumPy
Scikit-Learn
XGBoost
Matplotlib
Seaborn
Imbalanced-Learn (SMOTE)
Objective

The goal of this project is to accurately predict osteoporosis risk and support early healthcare decision-making through data-driven insights.
